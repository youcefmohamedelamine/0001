🤖 دليل إعداد بوت تيليجرام

بوت بيع الملفات مقابل نجوم تيليجرام ⭐

📋 المتطلبات الأساسية

1️⃣ تثبيت Python

تأكد من تثبيت Python 3.8 أو أحدث على جهازك

python --version

2️⃣ تثبيت المكتبات المطلوبة

قم بتثبيت مكتبة python-telegram-bot:

pip install python-telegram-bot --upgrade

🔑 الحصول على توكن البوت

خطوات إنشاء البوت

  • افتح تيليجرام وابحث عن @BotFather
  • أرسل الأمر /newbot
  • اختر اسماً للبوت (مثل: ملفاتي بوت)
  • اختر username للبوت (يجب أن ينتهي بـ bot)
  • ستحصل على التوكن، احتفظ به بسرية تامة!

⚠️ تحذير هام

لا تشارك التوكن مع أي شخص! من يملك التوكن يستطيع التحكم بالبوت بالكامل.

💳 تفعيل نظام الدفع بالنجوم

خطوات التفعيل

  • اذهب إلى @BotFather
  • أرسل الأمر /mybots
  • اختر البوت الخاص بك
  • اختر Payments
  • اختر Telegram Stars كوسيلة دفع
  • وافق على الشروط والأحكام

✅ ملاحظة

نجوم تيليجرام متاحة فقط للبوتات المعتمدة. تأكد من أن حسابك يدعم هذه الميزة.

⚙️ إعداد الكود

تعديل الكود

في ملف البوت، ابحث عن هذا السطر وضع التوكن الخاص بك:

BOT_TOKEN = "ضع_توكن_البوت_هنا"

يمكنك أيضاً تعديل:

  • PRODUCT_NAME: اسم المنتج
  • PRODUCT_PRICE: السعر بالنجوم
  • PRODUCT_DESCRIPTION: وصف المنتج
  • محتوى الملف في دالة successful_payment_callback

🚀 تشغيل البوت

خطوات التشغيل

  • احفظ الكود في ملف باسم telegram_bot.py
  • افتح Terminal أو Command Prompt
  • انتقل إلى مجلد الملف
  • شغل البوت بالأمر:
python telegram_bot.py

✨ البوت يعمل الآن!

اذهب إلى تيليجرام وابحث عن البوت الخاص بك وابدأ باستخدامه!

🎯 استخدام البوت

الأوامر المتاحة

  • /start - لبدء التفاعل مع البوت
  • /help - لعرض تعليمات الاستخدام

عملية الشراء

  • المستخدم يضغط على زر "شراء الملف"
  • يظهر له فاتورة الدفع
  • يدفع باستخدام نجوم تيليجرام
  • يستلم الملف فوراً بعد الدفع الناجح

💡 نصائح مهمة

نصائح للنجاح

  • احتفظ بنسخة احتياطية من التوكن
  • اختبر البوت جيداً قبل نشره
  • يمكنك استضافة البوت على سيرفر ليعمل 24/7
  • راقب السجلات (logs) لمتابعة أداء البوت
  • يمكنك إضافة قاعدة بيانات لتخزين معلومات المستخدمين

🔧 التخصيص المتقدم

أفكار للتطوير

  • إضافة منتجات متعددة بأسعار مختلفة
  • إنشاء نظام خصومات وعروض
  • إضافة لوحة تحكم للإحصائيات
  • دعم أنواع ملفات مختلفة (صور، فيديو، PDF)
  • نظام إحالة وعمولات
